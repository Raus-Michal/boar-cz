Boar-cz webové stavby

Boar-cz staví webové projekty. Stavění webových projektů je programování: prezentačních webových stránek, webových článků a webových aplikací.

Naše služby : postavíme profesionální webovou stránku, webový článek anebo webovou aplikaci podle požadavků zákazníka.

Značku Boar-cz produkuje jeden programátor, přesto se o Boar-cz hovoří jako o nás. Uznejte samy, že podepisování webových stránek vlastním jménem včetně titulů, není zrovna trendy.

Název Boar-cz je složenina anglického slova divočák a cz označuje zemi původu Czech Republic.

Logo Boar-cz znázorňuje znak internetu, který představuje rypák divočáka s kly a nosními dírkami.

Stavba webové stránky

Boar-cz kóduje webové stránky základními jazyky pro programování webových projektů. Na přání můžeme kódovat webový projekt ve frameworku Vue 3.

Při návrhu a realizaci validních, responzivních a rychlých webových stránek prosazujeme jednoduchost a přehlednost.

Jednoduchost, neznamená kódování primitivních webů, ale programování efektivních a rychlých webových stránek.

Přehlednost, pro nás znamená, vkládání do projektu pouze přínosných objektů. Nevkládáme do projektu objekty, které jsou pro návštěvníka pouze líbivé, ale v důsledku návštěvníkovi odebírají pozornost, právě od věcí podstatných.

  • HTML 5
  • CSS 3
  • JavaScript ES6
  • VUE 3
  • SVG-XML
  • PHP 7

Ve světě internetu je možné se setkat s weby, které jsou graficky líbivé, ovšem pomalé a nepřehledné.

Například šablonové webové stránky s množstvím kýčovitých fotografií z fotobank a rozsáhlého menu skrytého v hamburgeru.

Obdobné webové stránky návštěvníka nepotěší! Čeká na vykreslení obsahu webové stránky a proplétá se nepřehledným webem. Bohužel, se návštěvníkovi často zobrazují chyby v podobě malého nečitelného textu anebo textu se špatným kontrastem. Někdy se objeví i přetečení obsahu mimo box anebo mimo okno prohlížeče.

Nástroje na stavbu webové stránky

Jazyky webové stránky (HTML, JavaScript, CSS a PHP) kódujeme ve Visual Studio Code (VSC).

Obrázky SVG kreslíme v Inkscape a výsledný kód upravujeme. Na optimalizaci a úpravu rastrových obrázků používáme Adobe PhotoShop.

Výsledný kód webové stránky se ladí v DevTools.

GIT nám dává plnou kontrolu nad vývojem projektů a GitHub zálohuje naše projekty.

Při práci se serverem používáme Filezilla.

Používáme i doplňkové balíčky Node.js (glyphhanger) a umělou inteligenci chatGPT.

  • Visual Studio Code
  • Inkscape
  • Adobe PhotoShop
  • Filezilla
  • DevTools Edge/Chrome
  • GIT a GiTHuB

Boar-cz staví kód webových stránek. Kód nepíšeme v jedničkách a nulách, ale skládáme písmeno za písmenem, tag za tagem.

Pokud kreslíme SVG, výsledný kód dostatečně očistíme. V případě rastrových obrázků dostatečně pro web optimalizujeme.

Když si nevíme rady, obrátíme se na umělou inteligenci chatGPT, který nám s kódem zaručeně poradí.

Kontrola kvality webové stránky

Kvalita kódu je pro nás důležitá! Všechny webové stránky, které Boar-cz staví, jsou validní a v souladu s World Wide Web Consortium (W3C).

Na prvním místě je validita, za ní následuje rychlost načítání webu a nezapomínáme ani na SEO.

Bezchybné zobrazování, fungování a responzivitu našich webových projektů testujeme přímo na zařízeních různého druhu (telefony, tablety a počítače s operačním systémem iOS, Android a Windows).

Existuje více možností, jak vytvořit webovou stránku bez nutnosti programování.

Můžeme koupit šablonu webu, použít generátor webové stránky, web je možné vytvořit například i v Microsoft Wordu.

Kdo měl možnost vytvořit webovou stránku v něčem podobném, nám jistě dá za pravdu, že kvalita kódu těchto projektů je nevalná, rychlost načítání tristní, SEO mizerné, responzivita pokulhává a velikost webové stránky je gigantická.

Prezentační weby mohou zahrnovat od tvorby životopisu, osobního webu, prezentace živnosti až po prezentaci portfolia firmy. Pokud o sobě, své živnosti anebo firmě chcete dát ve světě internetu patřičně vědět, je prezentační webová stránka správnou volbou. V době digitálních technologií je prezentační web nejlepší možnost, jak oslovit velké množství lidí, budoucích zákazníku anebo obchodních partnerů.

Webové články patří spíše do blogů než na samostatnou webovou stránku. Nemusí tomu tak být! Pokud bude vašim záměrem oslovit čtenáře něčím lepším než pouhým textem, je webový článek zajímavým řešením a rozhodně změnou pro čtenáře. Nevěříte? Přesvědčit se můžete v projektu Pěstování konopí a trest, kde je homepage webovým článkem.

Webové aplikace přináší možnosti, jak uživatele upoutat a rychlím způsobem mu umožnit výpočet prakticky čehokoli. Jednoduchou a však jedinečnou webovou aplikací je Kalkulátor trestu za pěstování konopí. Webové aplikace mají ovšem svá omezení, jak je možné se přesvědčit v projektu Noční VLK.
